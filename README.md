# Logical
Logical is a Logic gate simulator, made to be a minimalistic, easy-to-use software. oh yeah, and its free.

I made this because i was looking for a good logic ciruit simulator, the closest one i could find was logicly. i names logical after this, because im kinda uncreative. so sue me. but i didnt want to pay $60 for this, and the web version didnt have save/load function. so i made one. in my opinion, it looks way better then logicly.

**controlls**

- left click on pins: connect wires

- right click on a gate/note/input pin: delete gate/note or clear that input wire

- Mouse wheel: zoom in/out

- Mouse right-drag: pan camera

- Mouse middle double-click: auto-center view (if enabled in settings)

- Left click empty space: open gate menu / place selected gate

- Drag with left mouse: move selected gates/notes

- Left click INPUT: toggle on/off

- Left click + hold BUTTON: on while held, off on release

- Double-click CLOCK: open clock menu (set time, start/stop)

- Double-click DELAY: open delay menu (set delay time)

- Tab: open/close settings

- Esc: new file (save prompt)

- Space: exit app (save prompt)

- Ctrl+S: save (choose Segment or Whole File)

- Ctrl+L: load file (segment files paste at cursor)

- Ctrl+Shift+V: paste segment file at cursor

- Ctrl+C: copy selected gates

- Ctrl+V: paste copied gates at cursor

- Ctrl+Z: undo
  
- Ctrl+Y: redo
  
- Backspace: delete selected
  
- R: rotate selected gate pin orientation clockwise
  
- T: rotate selected gate pin orientation counter-clockwise
  
- C (without Ctrl): clear all gates quickly
